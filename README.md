# HAM-Key
一个单按键摩斯码蓝牙键盘

## 简介

`HAMKey`是一个与摩斯码密切相关的键盘项目。旨在为业余无线电爱好者和对摩斯码感兴趣的人群提供一个便利的输入外设。

本分支为`HAMKey-Lite`的主仓库。
